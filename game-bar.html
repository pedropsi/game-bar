Puzzlescript Game Bar

About the Puzzlescript Game Bar

All Pedro PSI's puzzlescript games make use of a "game bar" with features such as a undo/reset buttons, level selector, a music on/off button and a fullscreen button. This is how it looks like:

How to use the Puzzlescript Game Bar in your site

Step 1: including the code

Include your game page the javascript source and the stylesheet:

  • The javascript source: GameBar.js
  • The stylesheet: GameBar.css

The javascript source must be loaded after the game loads completely, because a few of the original game functions are overwritten (mostly because of the level selector).

Step 2: selecting your game

In the javascript source

Ensure the javascript source can find your game, by editing the line (near the end): var gameElementID="#gameCanvas";. The selector named "#gameCanvas" should match the ID of your game canvas.

In the stylesheet

Likewise, ensure the stylesheet can find your game in fullscreen, by editing the selectors div:fullscreen #gameCanvas and div:full-screen #gameCanvas.

In case your game bar is misaligned with wih your game, please edit also the selector .gameContainer to match the parent container (outside) of your game canvas.

Step 3 (optional): adding music

To control music via the game bar, each music track should be added to the page inside <audio> tags, placed anywhere. If no audio tags are found, when pressing the music button, the message "Error: can't find the Jukebox" will be displayed.

If you wish to delete the music button, simply delete the line ButtonOnClickHTML("♫",'ToggleCurrentSong(this)'),.

Customisation tips

Changing the colours of the Puzzlescript Game Bar

Edit the colours in the :root element of the stylesheet to change everything at once.

Changing the logo image in the level selector

Easy way

Add a logo image named "logo.png" to the images folder in the root directory.

Hard way

Edit the function OpenBalloon in the javascript source, specifically by changing the path string "images/logo.png" to wherever your logo may be located.

Changing the "solved level" ★ symbol

For now, Ctrl+F to search for ★, which can be replaced by other symbols. More customisation options are planned.

FAQ

Can you use Puzzlescript Game Bar for free?

Yes, it's under a MIT license.

Can't the music button control music added via the youtube keyword in the prelude?

No, sorry.

Are all Puzzlescript Game Bar's features available elsewhere?

The fullscreen button is new. The undo/reset and music buttons are found in Puzzlescript's default mobile menu and Alan Hazelden has made a custom level selector. Nevertheless, Pedro PSI found it useful to have a single bar with all important features.

Does the Puzzlescript Game Bar level selector handle checkpoints?

Currently, yes, but only in games where among all levels, only a single level contains all checkpoints (e.g. a open world map). If multiple levels have checkpoints, it may not work correctly. See it in action in Tiaradventur.

Can the Puzzlescript Game Bar be extended with custom buttons?

Yes. Search for the function GameBar(targetIDsel) and edit the buttons array inside that function. In particular, try uncommenting the commented lines and see it change.

How about the "Credits" and "How-to-play" buttons?

You can either make your own or add the default ones (by uncommenting the lines referred to in the previous question). In the latter case you'll also need to have two sections in your page explicitly labelled with ids: Credits and How-to-play.
