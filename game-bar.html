Puzzlescript Game Bar

About the Puzzlescript Game Bar

All Pedro PSI's puzzlescript games make use of a "game bar" with features such as a undo/redo buttons, level selector, a music on/off button and a fullscreen button.

How to use the Puzzlescript Game Bar in your site

Step 1

Simply include your game page the javascript source and the stylesheet:

  • The javascript source: GameBar.js
  • The stylesheet: GameBar.css

The javascript source must be loaded after the game loads completely, because a few of the original game functions are overwritten (mostly because of the level selector).

Step 2

Ensure the javascript source can find your game, by editing the line (near the end): var containerID="#game";. The selector named "#game" should be replaced by a different selector matching the container (div) holding your game (canvas). Often this container matches the class selector ".gameContainer".

Step 3

Customise the stylesheet to agree with your page's style and game selectors.

Don't forget to replace selectors .game-container {etc...};, .game {etc...}; and #gameCanvas {etc...}; with the appropriate selectors for your page.

Problems? Suggestions? Success?

Either way, please let me know! I intend to improve this page so it is easier to use.

FAQ

Can you use Puzzlescript Game Bar for free?

Yes, it's under a MIT license.

Are the Puzzlescript Game Bar's features available elsewhere?

Some features can already be found in Puzzlescript's default mobile menu and Alan Hazelden 's level selector.

Does the Puzzlescript Game Bar level selector handle checkpoints?

Currently, only in games where among all levels, only a single level contains all checkpoints (e.g. a open world map). If multiple levels have checkpoints, it may not work correctly.
