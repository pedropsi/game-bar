Puzzlescript Game Bar

About the Puzzlescript Game Bar

This is the "game bar" found in all Pedro PSI's puzzlescript games. It features for instance undo/reset buttons, a beautiful level selector, a fullscreen button and an optional music button. Sounds useful? Well you too can use it!

How to add the Puzzlescript Game Bar to your own game

Step 1: include the code

Include the javascript source and the stylesheet in a regular Puzzlescript export or in a custom page:

  • The stylesheet: GameBar.css
  • The javascript source: GameBar.js

The stylesheet can be added to the <head> via <link rel="stylesheet" type="text/css" href="GameBar.css" media="screen"/ >.

The javascript source must be loaded after the game loads completely, because a few of the original game functions are overwritten (mostly because of the level selector). So you can add to the end of the <body> this <script type="text/javascript" src="GameBar.js">>.

Step 2: select your game (custom pages only)

In the javascript source

Ensure the javascript source can find your game, by editing the line (near the end): var gameSelector="#gameCanvas";. The selector named "#gameCanvas" should match the ID of your game canvas.

In the stylesheet

Likewise, ensure the stylesheet can find your game in fullscreen, by editing the selector #gameCanvas (first line)

In case your game bar is misaligned with wih your game, please edit also the selector .gameContainer to match the parent container (outside) of your game canvas.

Step 3 (optional): adding music

To control a music playlist via the game bar, each music track should be added to the page inside <audio> tags, placed anywhere. If no audio tags are found, when pressing the music button, the message "Error: can't find the Jukebox" will be displayed.

To reveal the music button (hidden by default), simply uncomment the line //ButtonOnClickHTML("♫",'ToggleCurrentSong(this)'), by deleting the //.

Customisation tips

Colours

Edit the colours variables in the :root element of the stylesheet.

Font

Edit the font variable (right after the colours) in the stylesheet.

Logo image beside the level selector

Easy way

Add a logo image named "logo.png" to the images folder in the root directory.

Hard way

Edit the function OpenBalloon in the javascript source, specifically by changing the path string "images/logo.png" to wherever your logo may be located.

"Solved level" ★ symbol

For now, Ctrl+F to search for ★, which can be replaced by other symbols. More customisation options are planned.

FAQ

Can you use Puzzlescript Game Bar for free?

Yes, it's under a MIT license.

How can the Puzzlescript Game Bar be used in the itch.io platform?

Please ensure that the Puzzlescript Game Bar sources are loaded from inside the <iframe> containing the game.

Are all Puzzlescript Game Bar's features available elsewhere?

The fullscreen button is new. The undo/reset and music buttons are found in Puzzlescript's default mobile menu and Alan Hazelden has made a custom level selector. Nevertheless, Pedro PSI found it useful to have a single bar with all important features.

What are the differences between the shared Puzzlescript Game Bar and the one live at the Creative Archive?

The live Puzzlescript Game Bar at the Creative Archive is the cutting-edge, sometimes experimental version. As soon as the changes are consolidated, they will be added to the official Puzzlescript Game Bar shared in this page. Any buttons that depend on the Creative Archive's infrastructure to work or that lack general interest won't be included into the official Puzzlescript Game Bar.

Does the Puzzlescript Game Bar level selector handle checkpoints?

Only in case all checkpoints are concentrated in a single level (e.g. a open world map such as in Tiaradventur). Support for multiple levels with checkpoints is not planned, unless you actually need it.

Why can't the music button control music added via the youtube keyword in the prelude?

Because this keyword is dubious.

How to extend the Puzzlescript Game Bar with custom buttons?

Search for the function GameBar(targetIDsel) and edit the buttons array inside that function. In particular, try uncommenting the commented lines and see it change.

How about the "Credits" and "How-to-play" buttons?

You can either make your own or add the default ones (by uncommenting the lines referred to in the previous question). In the latter case you'll also need to have two sections in your page explicitly labelled with ids: Credits and How-to-play.
