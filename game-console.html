Game Console

About the Game Console

This is just a convenient page to test any puzzlescript game right now with the game bar!

How to load any puzzlescript game in this page

Just replace the PUZZLESCRIPT_ID with your game's puzzlescript ID at the end of the following URL: https://pedropsi.github.io/game-console.html?game=PUZZLESCRIPT_ID.

Known issues

Sharing saved game progression

Game progression within the Game Console is not (currently) shared with the original game page. For the time being, please imagine two different game cartridges. Thanks J. C. Miller (jcGyo) for reporting.

FAQ

What is a puzzlescript id?

It's the long identifier after puzzlescript.net/play.html?p= or after puzzlescript.net/editor.html?hack=.

Why is this game bar slighly different from the published version?

Because this is the cutting-edge, possibly experimental, version!

Who may link to a game played in this page?

If you are the game author or obtained permission from the author, you may. Otherwise, please be kind and use instead the original links the authors provided, which help them monitor game traction and possibly boost their revenue.

If you really think a game should be played with the game bar, try to persuade the authors to add the game bar to their own game page.

Legal info

Particular games

All games playable in the Game Console are loaded in real time from a public github gist shared by the author, and not directly hosted nor verified by Creative Archive. Thus, if you are an author and want your game manually excluded from the Game Console, simply request it.
